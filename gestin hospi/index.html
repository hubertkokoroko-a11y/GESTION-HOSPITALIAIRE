Configuration Firebase requise

Pour que plusieurs personnes voient les mêmes données, vous devez configurer Firebase.

Étapes à suivre :

  1. Allez sur console.firebase.google.com
  2. Créez un projet et activez Firestore Database
  3. Copiez les clés de configuration
  4. Ouvrez le fichier index.html dans un éditeur de texte
  5. Remplacez le bloc firebaseConfig par vos clés
  6. Enregistrez et republiez sur GitHub
